# obsidian-knowledge

Curated knowledge base on AI agents, ecommerce, and prediction markets.

This is a subset of a personal Obsidian vault published as a standalone repo. Notes are interlinked using `[[wikilinks]]` — open the folder in [Obsidian](https://obsidian.md) for the best reading experience, or browse the markdown files directly on GitHub.

## Structure

- **Agents/** — Agent architecture, memory systems, orchestration, tooling, and background agents
- **AI Agents/** — Investment-focused multi-agent systems
- **Ecommerce/** — Marketing, creative testing, ad strategy, and automation
- **Prediction Markets/** — Arbitrage, quantitative trading, and market microstructure
- **Thinking/** — Mental models
- **_media/** — Images referenced by notes

See [[moc - Knowledge]] for a navigable index of all notes.

## Attribution

Most content in this vault is curated from public sources — X/Twitter posts, research papers, blog posts, and conference talks. Original authors are credited in each note's frontmatter (`source:` field) or inline attribution. This repo is a personal knowledge collection, not a claim of original authorship.
